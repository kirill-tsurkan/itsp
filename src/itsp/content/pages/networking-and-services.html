Рабочие станции и серверы на базе операционных систем Windows, Linux

Linux, FreeBSD — сервера с функциями маршрутизатора для подключения Интернет, контроль доступа офисной сети к интернет-услугам и управление доступом из интернета к сервисам офисной сети.

Linux — рабочие станции. Нет лицензий Windows, лучше защищенность от компьютерных вирусов. Есть ограничения по принтерам, сканерам, пользовательскому софту. Более строгие требования к аппаратным характеристикам.

Windows — централизованная авторизация пользователей, управление доступом к ресурсам сети (Active Directory). Файловые сервера, терминальные сервера (удаленный рабочий стол), 1С (всех версий), MS SQL Server (всех версий).

Используемые инструменты — VMWare, Citrix Xen, MS Hyper-V, Linux KVM. Выполняем работы по проектированию, поставке и дальнейшей эксплуатации комплексов виртуализации, в том числе с общим сетевым хранилищем и отказоустойчивой группой серверов с балансировкой нагрузки и резервированием сервисов.

Администрирование 1С

Технологическое обслуживание

Для поддержки оптимальной производительности баз 1С необходимо регулярно выполнять работы по оптимизации. Эти работы включают в себя такие операции как: проверка логической и ссылочной целостности с исправлением обнаруженных проблем; переиндексация базы и дефрагментация индексов; оптимизация работы сервера SQL и т.п.

Перевод в SQL формат

Можем обеспечить работу 1С в режиме базы SQL с помощью MS SQL Server или PostgreSQL (бесплатная лицензия). Возможны решения с серверами 1С и SQL на основе системы Linux (бесплатная лицензия). В таком сценарии Windows может использоваться только на некоторых рабочих местах в офисе, что позволяет значительно снизить затраты на лицензирование.

Обеспечение эффективной работы на серверах

Очень часто используемые сервера с функциями 1С и SQL не настроены на оптимальную производительность для этих сервисов. В числе типовых проблем обычно встречаются: неэффективное использование дисков, неоптимальная настройка SQL, базовые настройки производительности серверного оборудования, не соответствующие текущей нагрузке настройки программы сервера «1С Предприятие».

Организация резервного копирования и управление копиями

Исходя из требований Заказчика создаётся автоматизированная система для создания резервных копий баз данных в нужное время и с нужными параметрами. Также определяются сроки хранения полученных архивных файлов. Типовой сценарий — это создание бэкапа базы каждую ночь, создание бэкапа транзакций базы в течение дня, хранение бэкапов базы за последние 7 дней, а также хранение бэкапов на 1 число каждого месяца последние 12 месяцев. Также в состав услуги входит регулярная проверка возможности восстановить данные из имеющихся бэкапов.

Общее администрирование пользователей и организация доступа к сервису

Выполняем базовую поддержку работы пользователей с 1С — контроль зависших сеансов, первичная диагностика проблем в работе базы, анализ возможных причин блокировок и замедления операций, оперативное исправление аварий на уровне операционной системы и службы сервера 1С без модификаций в конфигурации.
